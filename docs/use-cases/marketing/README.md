- Align content with product launches
- Coordinate with sales initiatives

name: social-media-writer

# Social Media Writer Skill

## Instructions
1. Identify the platform and audience
2. Adapt tone and format for platform
3. Include platform-specific best practices
4. Add relevant hashtags
5. Include call-to-action
6. Suggest visual content if applicable

## Platform Guidelines

### LinkedIn (Professional)
- **Length:** 150-300 characters optimal
- **Tone:** Professional, insightful
- **Format:** Text + image or document
- **Hashtags:** 3-5 relevant tags
- **Best practices:**
  - Hook in first line
  - Use white space for readability
  - Tag relevant people/companies
  - Ask questions to drive engagement

### Twitter/X (Conversational)
- **Length:** 200-280 characters
- **Tone:** Conversational, timely
- **Format:** Text + image/video
- **Hashtags:** 1-3 tags
- **Best practices:**
  - Strong opening hook
  - Use threads for longer content
  - Engage with replies
  - Post at optimal times

### Instagram (Visual)
- **Caption:** 100-200 characters
- **Tone:** Authentic, inspiring
- **Format:** High-quality visual + caption
- **Hashtags:** 5-15 tags (mix of sizes)
- **Best practices:**
  - First line must grab attention
  - Use emojis appropriately
  - Include clear CTA
  - Story integration

### Facebook (Community)
- **Length:** 100-250 characters
- **Tone:** Friendly, engaging
- **Format:** Image/video + text
- **Hashtags:** 1-3 tags
- **Best practices:**
  - Ask questions
  - Share behind-the-scenes
  - Use native video

## Post Templates

### Announcement Post
```
📢 Exciting news! [Announcement]

[1-2 sentences on why this matters]

[Key benefit/feature]

Learn more: [Link]

#Hashtag1 #Hashtag2
```

### Educational Post
```
💡 Quick tip: [Tip headline]

[2-3 sentences explaining the tip]

[Example or use case]

Try it and let us know how it goes!

#Hashtag1 #Hashtag2
```

### Engagement Post
```
Question for [audience]: [Question]

[Context or why you're asking]

Drop your answer below! 👇

#Hashtag1 #Hashtag2
```

### Case Study/Success
```
🎉 Customer win: [Customer] achieved [result]

The challenge: [Brief description]
The solution: [How they used your product]
The result: [Quantified outcome]

Read the full story: [Link]

#Hashtag1 #Hashtag2
```

## Hashtag Strategy
- **Branded:** Your company/product tags
- **Industry:** Category and sector tags
- **Community:** Audience-specific tags
- **Trending:** Relevant trending tags (use carefully)

# SEO Optimizer Skill

## Instructions
1. Analyze content for target keywords
2. Optimize on-page elements
3. Improve content structure
4. Suggest internal linking
5. Generate meta tags
6. Check readability and quality

## On-Page SEO Checklist

### Title Tag
- [ ] Include primary keyword
- [ ] Keep under 60 characters
- [ ] Make it compelling (CTR optimization)
- [ ] Include brand name if relevant

### Meta Description
- [ ] Include primary keyword
- [ ] Keep 150-160 characters
- [ ] Write as ad copy (drive clicks)
- [ ] Include call-to-action

### Headers (H1, H2, H3)
- [ ] H1 includes primary keyword
- [ ] H2s include secondary keywords
- [ ] Logical hierarchy
- [ ] Descriptive and scannable

### Content Body
- [ ] Primary keyword in first 100 words
- [ ] Keywords used naturally (no stuffing)
- [ ] Related terms and synonyms included
- [ ] Comprehensive coverage of topic
- [ ] Original insights and value

### Internal Linking
- [ ] Link to relevant pillar content
- [ ] Link to related articles
- [ ] Use descriptive anchor text
- [ ] 2-5 internal links per post

### Images
- [ ] Descriptive file names
- [ ] Alt text with keywords
- [ ] Compressed for speed
- [ ] Captions where helpful

### URL Structure
- [ ] Short and descriptive
- [ ] Include target keyword
- [ ] Use hyphens for spaces
- [ ] No unnecessary parameters

## Keyword Integration

### Primary Keyword
- Use in: Title, H1, first paragraph, 2-3 H2s, conclusion
- Density: 1-2% natural usage

### Secondary Keywords
- Use in: H2s, body content
- Density: 0.5-1% each

### Related Terms
- Include naturally throughout
- Shows topic expertise to search engines

## Output Format

```markdown
# SEO Optimization Report: [Content Title]

## Target Keywords
- Primary: [keyword] (Volume: X, Difficulty: Y)
- Secondary: [keyword], [keyword]

## Meta Tags
**Title:** [Optimized title - X characters]
**Description:** [Optimized description - X characters]

## Recommended Changes

### High Priority
1. [Change with specific example]
2. [Change with specific example]

### Medium Priority
1. [Change with specific example]

### Suggestions
1. [Additional optimization ideas]

## Internal Linking Opportunities
- Link to: [Article title] with anchor "[anchor text]"
- Link to: [Article title] with anchor "[anchor text]"

## Content Gap Analysis
Competitors cover these topics we're missing:
- [Topic 1]
- [Topic 2]

## Readability Score
- Current: [Score]
- Target: [Score]
- Recommendations: [List]
```

## Workflow Examples

### Blog Post Production Workflow

```bash
# Step 1: Topic research
qwen -p "
Research blog topics for our B2B SaaS company.
Target audience: VP of Engineering at mid-market companies.
Themes: AI in software development, team productivity, 
technical debt management.
Find topics with search volume >500/month and 
medium competition.
" > blog-topics-q2.md

# Step 2: Create outline
qwen -p "
Create a detailed outline for: 'How to Reduce 
Technical Debt Without Slowing Down Development'
Include: H1, H2s, H3s, key points per section,
examples to include, and target word count (2000).
" > blog-outline.md

# Step 3: Write first draft
qwen -p "
Write the full blog post using @blog-outline.md
Tone: Professional but conversational
Include: Real examples, actionable tips, statistics
Target: 2000 words
" > blog-draft.md

# Step 4: SEO optimization
qwen -p "
Optimize @blog-draft.md for SEO:
Primary keyword: 'reduce technical debt'
Secondary: 'technical debt management', 'software development'
Generate meta title and description.
Suggest internal links to existing content.
"

# Step 5: Create social posts
qwen -p "
Create social media posts to promote the blog:
- 3 LinkedIn posts (different angles)
- 5 Twitter posts (including thread)
- 1 email newsletter blurb
Include relevant hashtags for each platform.
"
```

### Product Launch Campaign

```bash
qwen -p "
Create a comprehensive product launch campaign plan:

Product: AI-powered code review tool
Target: Engineering teams at tech companies
Launch date: April 15
Budget: $50,000

Include:

## Best Practices

### 1. Content Quality
- Fact-check all claims
- Include original insights
- Use data and examples
- Edit ruthlessly for clarity

### 2. Consistency
- Maintain publishing schedule
- Consistent brand voice
- Regular performance reviews
- Continuous optimization

### 3. Distribution First
- Plan distribution before creation
- Multi-channel approach
- Paid amplification for best content
- Employee advocacy program

### 4. Measurement
- Define success metrics upfront
- Track full funnel impact
- Attribute revenue to content
- Regular performance reviews

### 5. Iteration
- Refresh top-performing content
- Double down on what works
- Kill underperforming initiatives
- Test new formats and channels
