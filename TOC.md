## About Qwen Code

### Philosophy
- Open source and co-evolving with the Qwen3-Coder model
- Terminal-first design with IDE-friendly integration
- Rich agentic workflow capabilities
- Unix philosophy: composable and scriptable

### Key Features Summary

| Feature | Description |
|---------|-------------|
| **Skills** | Model-invoked modular capabilities |
| **Sub-Agents** | Specialized AI assistants |
| **Approval Modes** | Four permission levels for safety |
| **MCP** | External tool and data source integration |
| **Commands** | Slash, @, and ! command system |
| **Headless** | Scriptable automation for CI/CD |
| **IDE Integration** | VS Code, Zed, JetBrains support |
